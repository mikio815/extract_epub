# ⚠️絶対に本体ファイルの中身を読んでから、かつ自己責任で利用してください！！！⚠️

ファイルサーバーから落としてきたフォルダ内のepubファイルをすべてzipにリネームして、かつ/OEBPS/imageフォルダを/に引き上げるシェルスクリプトです。

##　使い方

### 1. パスを書き換える
extract_epub.ps1ファイルをメモ帳などで開き、"$rootDir = "pass""のpassの部分を展開したいフォルダのパスに書き換えます。（例えば、"C:\Users\s28887\Documents\ghost_in_the_shell\sozai"）**このとき、パスに日本語が入らないようにしてください。**  
  
### 2. スクリプトを実行可能にする
管理者権限でpowershellを開き、
```
Set-ExecutionPolicy Unrestricted -Scope Process
```
を実行し、powerシェルスクリプトを実行可能にします。
  
### 3. 実行する
extract_epub.ps1を実行するとpass配下に1, 2, 3, ...と巻数順にリネームされたフォルダが作成されます。
